# Paper Assist for Chrome (확장 프로그램)

현재 보고 있는 브라우저 탭을 캡처해서, **로그인된 Claude / ChatGPT 구독 계정**에 자동으로 이미지+프롬프트를 넣고 전송하는 Chrome 확장입니다. 별도 API 키가 필요 없습니다.

## 설치 방법

1. Chrome 주소창에 `chrome://extensions` 입력 후 이동
2. 오른쪽 위 **개발자 모드(Developer mode)** 켜기
3. **압축해제된 확장 프로그램을 로드합니다(Load unpacked)** 클릭
4. 이 폴더(`chrome_extension`)를 선택
5. 도구모음에 퍼즐 아이콘 ▸ Paper Assist 를 고정(핀)하면 편합니다

## 사용법

1. 분석할 논문/문서/그래프가 있는 탭을 화면에 띄웁니다.
2. **Paper Assist 아이콘** 클릭 → 팝업이 열립니다.
3. **Claude** 또는 **ChatGPT** 선택 → 프리셋 클릭 또는 프롬프트 직접 입력.
4. **현재 탭 캡처 & 분석 요청** 클릭.
5. 새 탭에서 해당 서비스가 열리고, 캡처 이미지와 프롬프트가 자동 입력 후 전송됩니다. 답변은 그 탭에서 확인하세요.

> 처음 사용 시 claude.ai / chatgpt.com 에 **로그인(구독)** 되어 있어야 합니다.

## 동작 방식

- `chrome.tabs.captureVisibleTab` 으로 현재 탭의 보이는 영역을 PNG로 캡처
- 대상 사이트를 새 탭으로 열고, 콘텐츠 스크립트가:
  - 숨겨진 파일 입력(`input[type=file]`)에 이미지를 첨부 (실패 시 붙여넣기 이벤트로 폴백)
  - 입력창(contenteditable / textarea)에 프롬프트 삽입
  - 전송 버튼 클릭 (못 찾으면 Enter 로 시도)

## 주의 / 한계

- **사이트 UI가 바뀌면 동작이 깨질 수 있습니다.** 그럴 때는 `content.js` 상단의 `SELECTORS` 값(입력창·파일입력·전송버튼 셀렉터)만 수정하면 됩니다.
- 캡처는 **현재 탭의 보이는 영역 전체**입니다. (영역 드래그 선택은 포함되지 않음 — 필요하면 추가 가능)
- 자동 첨부가 실패하면 이미지를 클립보드에 복사하고 안내 메시지를 띄웁니다. 그 경우 입력창에서 ⌘V 로 직접 붙여넣으세요.
- 이 방식은 사용자 본인의 로그인 세션에서 브라우저를 자동 조작합니다. 각 서비스 이용약관을 준수해 사용하세요.

## 파일 구조

```
chrome_extension/
├── manifest.json   # 확장 정의 (MV3)
├── popup.html      # 팝업 UI
├── popup.js        # 팝업 로직
├── background.js   # 캡처 + 탭 열기 + 스크립트 주입
└── content.js      # 대상 사이트에서 이미지/프롬프트 주입 + 전송
```
